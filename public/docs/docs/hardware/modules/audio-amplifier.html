Control de Ganancia
  • El pin 1 y 8 son los pines de ganancia.

Sí se cierra el circuito entre ambos pines se setea una ganancia de 20 (26dB) porque internamente se conecta una resistencia de 1.35kOhm.
Sí se pone un capacitor entre ambos pines se setea una ganancia de 200(46dB).

Polarización de Entrada
  • Se explican muchas situaciones donde puede haber un offset no deseado y como solucionarlo
  1. En caso de utilizar una señal muy pequeña en los input y solo utilizar uno, se recomienda conectar el otro input a tierra para no tener un offset indeseado. En caso de ser muy alta, no.
  2. Sí en cada input se conecta un capacitor no habrá un offset indeseado (obviamente).
Importante

En caso de tener una alta ganancia[1] la entrada no usada puede generar inestabilidades y ruido; un capacitor de 0.1 uF podría solucionarlo (o cortocircuitarla a tierra).
